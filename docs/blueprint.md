# Blueprint

## Purpose
Define the core product direction, user journeys, and success criteria for the Digital Seva App.

## Actors
- Citizen
- Service center operator
- Admin

## Core flows
1. Document intake and upload
2. Service request creation
3. Status tracking and notifications
4. Secure chat
5. Staged payments

## Success metrics
- Request creation time under target
- Document upload success rate above threshold
- Chat resolution rate
- Payment completion rate
